== Description ==
WooCommerce advance reporting is a comprehensive and the most complete reporting system of all past orders with graphic interactive representation. Data can be viewed starting from all years to all days in a month.

This plugin is ready to use with woocommerce.

== Installation ==

 Quick Setup


 1) Install plugin through backend of your Wordpress site or via FTP.

 Having trouble with installation? Just check our website with free videos on installation and setup.

 From Wordpress Backend (automatic):
 Go to `Plugins -> Add new` -> click on Upload Plugin and upload the File woocommerce-advance-reporting.zip,
 or by Search Plugins, type in:`woocommerce-advance-reporting` and click on ìnstall`

 Manual Setup:
 1. Upload `woocommerce-advance-reporting` Folder from unzip woocommerce-advance-reporting.zip to the `/wp-content/plugins/` directory
 2. Activate the plugin through the 'Plugins' menu in WordPress
